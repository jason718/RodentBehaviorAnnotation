## Setup
        
### Prerequisites
        - Linux
        - NVIDIA GPU + CUDA CuDNN Recommended(CPU mode and CUDA without CuDNN also work)
        
### Getting Started
        - Install [[Caffe]](http://caffe.berkeleyvision.org/)
        - Install [[pythonVideoAnnotator]](pythonVideoAnnotator)
        - Clone this repo
